Want to help?

DataSync is designed to accept contributions from the community. Interested in helping out? We've got a number of ways you can help improve our resources.

Found a bug?

If you've found a bug, please file an issue on Github and thank you so much if you do. We'll take a look and do our best to patch it up quickly!

Have a feature request?

If you have suggestions for new DataSync features, please feel free to file for an enhancement on Github. We'll take a look and see what we can do!

Think you can fix/implement it yourself?

If you've found an improvement you'd like to make yourself, send us a pull request and we'll happily review it. There are instructions on how to build in our README.
